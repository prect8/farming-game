# 農家の物語 - ビジュアルノベル

2025年11月29日、冬の始まり。都会から来た青年が祖父の農場を受け継ぎ、新しい人生を始める物語。

## 🎮 ゲーム概要

- **ジャンル**: ビジュアルノベル / 恋愛シミュレーション
- **プレイ時間**: 10-15分
- **エンディング**: 2種類（ハッピー、ノーマル）
- **選択肢**: 3箇所
- **対応ブラウザ**: Chrome, Firefox, Edge, Safari（最新版）

## 📖 ストーリー

都会の喧騒を離れ、祖父が残した古い農場へとやってきた主人公・ユウキ。
荒れ果てた農場を前に不安を感じていたところ、隣の牧場で働く明るい女性・リサと出会う。
彼女や村の人々との交流を通じて、新しい人生を切り開いていく――。

## ✨ 特徴

### ゲームシステム
- **選択肢分岐**: プレイヤーの選択でストーリーが変化
- **マルチエンディング**: 選択によって異なる結末
- **タイプライター効果**: テキストが1文字ずつ表示
- **オート再生**: 自動でストーリーが進行
- **スキップ機能**: 既読シーンを高速スキップ

### UI/UX
- **ギャラリー**: 既読CGを鑑賞可能
- **設定画面**: 音量・テキスト速度を調整
- **レスポンシブ対応**: PC・タブレット・スマホで動作
- **アニメーション**: スムーズな画面遷移

### オーディオ
- **BGM**: シーンに応じた音楽（3曲）
- **効果音**: クリック音、選択音
- **音量調整**: BGM・効果音を個別に調整可能

## 🚀 起動方法

1. `index.html`をブラウザで開く
2. 「はじめから」をクリック
3. ゲームスタート！

## 🎯 操作方法

### メニュー画面
- **はじめから**: 新規ゲーム開始
- **ギャラリー**: 既読CG鑑賞
- **設定**: 音量・テキスト速度調整

### ゲーム画面
- **クリック**: 次のシーンへ進む
- **タイピング中にクリック**: 全文即座表示
- **選択肢**: ボタンをクリックして選択

### コントロールボタン
- **📋 メニュー**: メニュー画面に戻る
- **▶️ オート**: 自動再生ON/OFF
- **⏩ スキップ**: 既読シーンを高速スキップ

## 🐛 トラブルシューティング

### 音声が再生されない
- 音声ファイルが正しいフォルダに配置されているか確認
- ブラウザの音量設定を確認
- ファイル名が正しいか確認（menu.mp3, main.mp3, ending.mp3）

### 画像が表示されない
- 画像ファイルが`assets/images/`に配置されているか確認
- ファイル名が正しいか確認
- ブラウザのキャッシュをクリア（Ctrl+Shift+R）

### ゲームが進まない
- ブラウザのコンソール（F12）でエラーを確認
- JavaScriptが有効になっているか確認

## 📝 カスタマイズ

### ストーリーの編集
`story.js`を編集してストーリーを変更できます。

### 画像の追加
`assets/images/`に画像を追加し、`story.js`で指定します。

### 音声の追加
`assets/audio/bgm/`または`assets/audio/sfx/`に音声ファイルを追加します。
