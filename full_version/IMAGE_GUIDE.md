# 画像管理ガイド

## 画像の種類

### 背景画像
- 農場の風景
- 村の風景
- 季節や時間帯の違い

### キャラクター画像
現在使用中：
- `00001-3689628275.png` - リサ（ヒロイン）
- `00004-2642426617.png` - タクミ（大工）
- その他の画像は背景として使用

## 画像の命名規則

### 推奨される命名（将来的に整理する場合）

#### 背景
```
bg_farm_winter_day.png      - 農場（冬・昼）
bg_farm_winter_evening.png  - 農場（冬・夕方）
bg_village_festival.png     - 村の祭り
```

#### キャラクター
```
char_lisa_normal.png        - リサ（通常）
char_lisa_smile.png         - リサ（笑顔）
char_takumi_normal.png      - タクミ（通常）
char_hana_normal.png        - ハナばあちゃん（通常）
```

## 画像の配置

すべての画像は以下のフォルダに配置：
```
full_version/assets/images/
```

## 画像の推奨サイズ

- **背景**: 1920x1080px（16:9）
- **キャラクター**: 高さ1000-1500px（立ち絵）
- **フォーマット**: PNG（透過対応）

## 画像の追加方法

1. 画像を `full_version/assets/images/` に配置
2. `story.js` のシーンデータで指定
   ```javascript
   {
       id: 'scene_xxx',
       bg: '新しい背景.png',
       char: '新しいキャラ.png',
       // ...
   }
   ```

## 最適化のヒント

- 画像サイズを適切に圧縮（TinyPNG等を使用）
- 不要な画像は削除
- 同じ画像を複数回使い回す
- WebP形式も検討（ブラウザ対応確認）
